# PRISM Project Website

Static GitHub Pages project website for **PRISM: Precision and contact-rich Real-world Industrial Skill Dataset with Multimodal Sensing**.

Open `index.html` directly or serve this directory with:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

Resource links for arXiv, ModelScope, GitHub, and video are placeholders until the final URLs are available.
